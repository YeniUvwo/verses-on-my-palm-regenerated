VERSES ON MY PALM
Static launch build

UPLOAD TO GO54
1. Create or open the domain's document root in cPanel File Manager.
2. Upload every file and folder from this package.
3. Keep index.html, manifest.webmanifest and service-worker.js at the root.
4. Activate SSL and open the site using https://.
5. If replacing an older build, clear browser cache or update the cache name in service-worker.js.

TECHNICAL
- No database
- No user accounts
- Saved verses use localStorage
- Search runs entirely in the browser
- Installable PWA
- Offline shell caching
- KJV Scripture excerpts
